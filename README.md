# Pino

A collection of Toram Online formulas.

> still in development!

### Install

```
npm i @jmmaa/pino@0.0.2-dev
```

### Example

```typescript
import * as pino from "@jmmaa/pino";

const level = 275;
const totalVIT = 1;

pino.baseMaxHP(level, totalVIT); // 2238
```
